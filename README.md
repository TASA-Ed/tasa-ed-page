# TASA-Ed 官网

使用 Svelte 5。

## 配置项目

```bash
git clone https://github.com/TASA-Ed/tasa-ed-page.git
```

### 安装依赖

```bash
pnpm i
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

### 检查

```bash
pnpm check
```

### 清除缓存

```bash
pnpm prepare
```

### Lint

```bash
pnpm lint
```
